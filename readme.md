## Development Outline

#### HTML & CSS
Adopt `BEM CSS` naming convention for `html` and `css` files

#### Javascript
Name functions with `verb+x` convention. eg. `updatePics()`, `setMode()`
where `x` is noun, etc
#### General Guidelines
* Let namings be brief and clear
* Ensure readability, format files always
* Refactor on the go
* Be consistent
